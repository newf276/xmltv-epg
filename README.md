# xmltv-epg

Provide daily EPG for selected websites using [iptv-org/epg](https://github.com/iptv-org/epg)

### Currently supported websites:

| Websites |                  English                 |
|----------|:-----------------------------------------:|
|   Pluto  |  [xml][pluto-en-xml] - [gzip][pluto-en-gzip] |
|   Plex   |  [xml][plex-en-xml] - [gzip][plex-en-gzip] |

[pluto-en-xml]: https://github.com/newf276/xmltv-epg/releases/latest/download/epg-pluto-en.xml
[plex-en-xml]: https://github.com/newf276/xmltv-epg/releases/latest/download/epg-plex-en.xml
[pluto-en-gzip]: https://github.com/newf276/xmltv-epg/releases/latest/download/epg-pluto-en.xml.gz
[plex-en-gzip]: https://github.com/newf276/xmltv-epg/releases/latest/download/epg-plex-en.xml.gz
